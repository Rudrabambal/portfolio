# 🚀 Rudra Bambal | Modern Developer Portfolio

A sleek, premium, and fully responsive developer portfolio website built using **Next.js**, **Tailwind CSS**, and **Framer Motion**. It features interactive canvas animations, modular section designs, and contact form integration.

🔗 **Live Link:** [https://portfolio-lj1i-a99gxhhvd-rudrabambal912-6901s-projects.vercel.app](https://portfolio-lj1i-a99gxhhvd-rudrabambal912-6901s-projects.vercel.app)

---

## ✨ Features

- **Dynamic Visuals:** Includes an interactive Starfield background component powered by HTML5 Canvas.
- **Beautiful Animations:** Smooth transition effects, page entries, and hover interactions powered by **Framer Motion**.
- **Contact Integration:** Functional contact form integrated with **EmailJS** for instant email alerts.
- **Clean Structure:** Modular section-based architecture for easy maintenance and scalability:
  - 🌟 **Hero:** Striking entrance header with call-to-actions.
  - 🧑‍💻 **About & Skills:** Interactive grid showcasing core competencies.
  - 🎓 **Education & Certifications:** Structured academic timeline and achievements.
  - 💼 **Experience:** Interactive career timeline displaying roles and descriptions.
  - 📁 **Projects:** Card layout featuring work highlights with clean hover animations.
- **Responsive Layout:** Optimized for mobile, tablet, and desktop screens.
- **Next-gen Tech Stack:** Leveraging Next.js App Router, Tailwind CSS, TypeScript, and Lucide React icons.

## 🚀 Getting Started

To run this project locally, follow these steps:

### 1. Clone the repository
```bash
git clone https://github.com/Rudrabambal/portfolio.git
cd portfolio
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the development server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### 4. Build for Production
To build and optimize the application for production deployment:
```bash
npm run build
npm run start
```
